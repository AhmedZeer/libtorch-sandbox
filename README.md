## Libtorch Sandbox

* This repo includes some Libtorch examples for learning purposes.
* Sum important commands are given.


init CMake : 
```sh
mkdir build
cd build

# Change the path
cmake -DCMAKE_PREFIX_PATH=/home/ahmed4/gitS/libtorch ..
cmake --build . --config Release
```

build :
```sh
cmake --build . --config Release
```
